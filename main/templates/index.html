Multilingual Cyberbullying Monitoring Application was designed to observe the pattern of cyberbullying on social public media like Twitter. This is a Python Webcrawler & Sentiment Analysis System.

Works for Igbo, English and Pidgin tweets or combination of the 3

Built with Python 3.8, Django 3, Vader sentiment library, snscrape library, Pandas, Seaborn, dbsqlite3, Googletrans and Python nltk library
